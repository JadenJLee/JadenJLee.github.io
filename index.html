Analyzing A College Messaging System

A group of my collegues created a graph data structure to assist in running algorithms on a CollegeMsg temporal network. We used this dataset to look at how students at UCI interact with each other through this messaging service, implementing algorithms that can tell us unique information about each of the students. We used a depth-first search (DFS) traversal recursively to traverse the dataset. We used Kruskals Algorithm to detect vertex (students) relationships to one another and with that information, build minimum spanning trees. Using this, we can find the miniimum number of friends that two users must interact with in order to become friends themselves. We also used PageRank algorithm to find the most "popular" people in this specific social network. We also used the linear algebra C++ library Armadillo for computing the PageRank algorithm and ranking popularity of people on the social network. Analyzing A College Messaging System on Github
